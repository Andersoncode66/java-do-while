# 💻Roteiro de Estudos
## 🖥️Básico
| Exercício | Tema  | Descrição |
| :---------------- | :--------- | :--------- |
| `01` | `Mostre números de 1 a 10.` | `Use um do-while para exibir todos os números de 1 a 100.` |
| `02` | `Mostre números de 10 até 1.` | `Mostre somente os números pares, com do-while` |
| `03` | `Mostre apenas números pares de 1 a 20.` | `Mostre somente os ímpares.` |
| `04` | `Mostre apenas números ímpares de 1 a 20.` | `Mostre os números de 100 até 0.` |
| `05` | `Some os números de 1 a 10.` | `Leia um número e mostre sua tabuada de 1 até 10 usando do-while.` |
| `06` | `Mostre a tabuada de um número informado pelo usuário.` | `Calcule a soma total usando do-while.` |
| `07` | `Leia um número e conte até ele.` | `Some somente os pares.` |
| `08` | `Leia um número e conte dele até 0.` | `Some somente os ímpares` |
| `9` | `Peça números até o usuário digitar 0.` | `Use o do-while para ler os 10 valores.` |
| `10` | `Conte quantos números foram digitados até o 0.` | `Armazene o maior valor encontrado.` |
